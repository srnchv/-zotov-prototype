# Цифровой архив Центра «Зотов» — клик-прототип

UX-прототип публичной части цифрового архива (Charmer studio).
Статический SPA: `index.html` + `style.css` + `app.js`, hash-роутинг, без сборки.

## Запуск локально
```
python3 -m http.server 5050 --directory .
```
Открыть http://localhost:5050

## Что внутри
Публичные сущности (материал, личность, место, событие, тема, проект/выставка,
организация, коллекция, источник, медиафайл, тег), разделы (архив, хронограф,
карта, каталоги), расширенный поиск, личный кабинет и флоу заявки на доступ.

Связи между сущностями двусторонние (строятся из данных) — демонстрируют
принцип «создано один раз → выводится во всех релевантных разделах».
